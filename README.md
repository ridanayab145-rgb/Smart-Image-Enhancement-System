
# Smart Image Enhancement System

## Description
A Python-based GUI system for **interactive image enhancement**, integrating multiple lab tasks from Digital Image Processing:
- Sampling & Quantization
- Geometric Transformations
- Intensity Adjustments
- Contrast Enhancement & Histogram Analysis

This system allows students to **upload images, apply multiple enhancements, visualize results side-by-side, and save outputs**.

---

## Features

- Upload image and view metadata (resolution, channels, data type)
- Sampling: adjust resolution scale
- Quantization: reduce bit depth
- Rotation: arbitrary angle rotation
- Negative transformation
- Gamma correction (brightness adjustment)
- Histogram equalization
- RGB histogram visualization
- Save enhanced images
- Modern GUI with scrollable sidebar and grey-themed buttons

---

## Technologies Used

- Python 3
- OpenCV
- Tkinter
- PIL / Pillow
- Matplotlib
